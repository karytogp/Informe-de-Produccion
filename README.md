# 📊 Production & Orders Dashboard | Power BI Project

## 📌 Descripción
Este proyecto consiste en un dashboard de producción y órdenes desarrollado en Power BI, como parte de un proceso de formación práctica en análisis de datos.

El objetivo fue construir una solución completa que permitiera monitorear indicadores operativos, analizar el comportamiento de las órdenes y entender factores que afectan el cumplimiento.

## 🎯 Objetivo
Analizar la relación entre producción y órdenes, identificar patrones en el tiempo y detectar oportunidades de mejora en la operación.

## 🛠 Herramientas utilizadas
- Power BI (Power Query, DAX)
- Excel

## 📊 Análisis realizado

### 🔹 KPIs principales
- Total Producción: 55,723
- Total Órdenes: 2,223
- % de Órdenes Entregadas

### 🔹 Análisis temporal
- Producción y órdenes por año
- Tendencias mensuales

### 🔹 Segmentación
- Tipo de cliente: Distribución, Directo, Otros
- Tipo de envío: Express, Regular, Terrestre
- Prioridad: Alta, Media, Baja

### 🔹 Análisis operativo
- Desempeño por gerente (Ana, Carlos, José, René)
- Evaluación de cumplimiento de entregas

### 🔹 Devoluciones
- Análisis de causas:
  - Contenedor dañado
  - Producto defectuoso
  - Entrega fuera de tiempo
  - Otros

## 📈 Insights clave
- Identificación de variaciones entre producción y órdenes a lo largo del tiempo
- Detección de factores que afectan el cumplimiento de entregas
- Visibilidad del desempeño por responsable (gerentes)
- Análisis de causas principales de devoluciones
